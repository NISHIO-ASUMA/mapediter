試作モデルビューワー

#-----------------------------
#model.txtに書くこと
#-----------------------------
1.Xファイルの種類数
2.読み込むXファイルのパス ( コメントの書き方は同じ書き方で! )
3.配置する際の移動キーの移動量
4.プレイヤーのジャンプ量

#------------------------------
#操作方法
#------------------------------

[ 移動 ] W/A/S/D
[ 配置 ] Enterキー
[ 種類切り替え ] ↑ / ↓
[ X方向の拡大率 ] Y / H
[ Y方向の拡大率 ] U / J
[ Z方向の拡大率 ] I / K
[ Y座標の増減   ] T / G
[ 配置物の消去  ] BACKSPACEキー ( 配置数が0以上の時)

[ ファイルに書き出し ] F7キー
[ 書き出すファイル切り替え ] F9キー
[ 配置物の再読みこみ ] F10キー
[ モード切替 ] F1 / F2 キー
[ ワイヤーフレーム ] F3 / F4 キー
